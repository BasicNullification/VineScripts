# UserScripts

These are userscripts that add functionality to Amazon Vine pages.

## Installation

### Tampermonkey
1. Install [Tampermonkey](https://www.tampermonkey.net/) for your browser
2. Click on any `.user.js` file in this folder → GitHub will show a raw view
3. Tampermonkey will detect it and show an install prompt
4. Click "Install" to add the script to Tampermonkey

For detailed instructions, see [Tampermonkey's installation guide](https://www.tampermonkey.net/).

### Greasemonkey
1. Install [Greasemonkey](https://www.greasespot.net/) for Firefox
2. Click on any `.user.js` file → Greasemonkey will show an install dialog
3. Click "Install" to add the script

For detailed instructions, see [Greasemonkey's documentation](https://www.greasespot.net/pages/script-installation).

## Updates
Scripts are configured for auto-updates. Your script manager will periodically check for new versions.
